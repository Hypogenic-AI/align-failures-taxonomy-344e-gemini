# Downloaded Datasets

This directory contains datasets for analyzing scientific misconduct and AI failures.

## Dataset 1: Retraction Watch Data
- **Source**: [GitLab (Crossref)](https://gitlab.com/crossref/retraction-watch-data)
- **Size**: 64 MB (CSV)
- **Task**: Thematic analysis of research failures and misconduct.
- **Location**: `datasets/retraction-watch-data/`

## Dataset 2: AI Incident Database (AIID)
- **Source**: [GitHub (RAIC)](https://github.com/responsible-ai-collaborative/aiid)
- **Size**: ~35 MB
- **Task**: Analysis of real-world AI failures and harms.
- **Location**: `datasets/aiid/`
